# Production-Ready ML System – Sales Prediction Project

This project is a Production-Ready Machine Learning System developed using Python and Scikit-learn for predicting sales based on advertising budget.

## Features

* Data Preparation using Pandas
* Linear Regression Model Training
* Model Saving & Loading using Joblib
* Sales Prediction System
* Data Visualization using Matplotlib

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Joblib

## Output

The model predicts future sales values based on advertising input and visualizes the relationship between advertising budget and sales using a line graph.

## Run the Project

```bash
pip install pandas matplotlib scikit-learn joblib
python main.py
```
